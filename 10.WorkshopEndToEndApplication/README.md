- [x] Vanilla JS

- [x] Lit-html for templating
- [ ] Mocha for unit testing
- [ ] Playwright for e2e testing
- [x] Vite Local dev server
- [x] Vite Bundling (resolve package names)
- [x] Git Source Control with GitHub
